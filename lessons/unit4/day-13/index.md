# Day 13: Biotechnology Tools and Applications

## Objectives

- Describe PCR, gel electrophoresis, recombinant DNA, GMO applications, and CRISPR basics.
- Evaluate biotechnology scenarios using scientific evidence.
- Distinguish between a claimed application and the data needed to support it.

## Key Terms

- PCR
- gel electrophoresis
- recombinant DNA
- genetic engineering
- GMO
- CRISPR
- biotechnology application

## Textbook Content

- Chapter 13 introduction to core biotechnology tools and applications.
- Basic structure of a biotechnology explanation: tool, target, evidence, and outcome.
- Real-world use cases that prepare students for data interpretation.

## In-Class Activities / Minilabs

1. **Bellwork (8 min):** Match a biotechnology tool to the kind of problem it helps solve.
2. **Mini-Lesson (15 min):** Model PCR, gel electrophoresis, recombinant DNA, and CRISPR at the "what it does" level.
3. **Guided Practice Block 1 (25 min):** Analyze biotechnology scenarios and identify which tool fits each use case best.
4. **Guided Practice Block 2 (27 min):** Label a simple biotechnology workflow and identify what evidence would show that each step worked.
5. **Debrief (10 min):** Write one evidence-based claim about a biotechnology application.

## Classroom Routine Notes

- Groups must identify the tool and expected evidence before they discuss ethical or practical concerns.
- The teacher uses a fixed role structure: reader, evidence finder, recorder, reporter.
- Students stay focused on what each tool does and what evidence would matter before diving into debate.

## Critical Thinking

- If a school, farm, lab, or clinic were considering a biotechnology tool, what evidence should they ask for before deciding to use it?
- How might the same biotechnology tool be useful in one real context but a poor fit in another?

## Biblical Integration

Biotechnology should be approached as stewardship that requires wisdom, humility, and careful evidence.

## Homework / Assignment

- Complete the biotechnology tool-and-application analysis questions.
- Write one short claim-evidence-reasoning response about a biotechnology example.

## Exit Ticket / Reflection

Which biotechnology tool feels clearest right now, and what evidence helped you understand it?

## Teacher Notes Only (not for students)

- Keep the chapter launch evidence-driven rather than turning it into a full ethics seminar.
- Save strong tool-and-evidence examples for Days 14 and 15.
