# Unit 4 Day 9 — Biotechnology Data Analysis

## Objectives (Tested)

- Interpret PCR output data
- Interpret gel electrophoresis band patterns
- Compare evidence from two biotechnology scenarios
- Write a claim-evidence-reasoning response using provided data

## Activities

1. PCR and gel data warm-up
2. Small-group data interpretation set
3. Claim-evidence-reasoning writing task

## Assessment (Formative)

- Data analysis check for accuracy and reasoning quality

## Homework (Formative)

- Correct missed data interpretation items and complete extension practice

## Teacher Notes Only (not for students)

- Connect precision and information flow in genetics to a design-centered worldview during discussion.
- Do not include worldview language in student assessments.
