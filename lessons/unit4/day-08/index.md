# Day 8: Complex Inheritance Patterns

## Objectives

- Explain incomplete dominance, codominance, and multiple alleles.
- Model blood-type inheritance using genotype evidence.
- Compare Mendelian and non-Mendelian outcomes using data.

## Key Terms

- incomplete dominance
- codominance
- multiple alleles
- ABO blood group
- inheritance model

## Textbook Content

- Chapter 12 coverage of non-Mendelian inheritance patterns.
- ABO blood group as a case study in multiple alleles.
- Evidence-based comparison of models rather than memorized labels.

## In-Class Activities / Minilabs

1. **Bellwork (8 min):** Differentiate genotype and phenotype with one non-Mendelian example.
2. **Mini-Lesson (15 min):** Model incomplete dominance, codominance, and multiple alleles with worked examples.
3. **Guided Practice Block 1 (27 min):** Complete a blood-type inheritance simulation and annotate genotype-to-phenotype links.
4. **Guided Practice Block 2 (25 min):** Correct one flawed complex inheritance response and compare which model fits each scenario best.
5. **Debrief (10 min):** Write one “watch out” note for complex inheritance problems.

## Classroom Routine Notes

- Students work in triads when possible: solver, checker, and vocabulary monitor.
- The teacher uses a fast reset cue between simulation and clinic work so materials stay organized.
- Students must name the evidence for a model before they label it.

## Critical Perspective

- Why is it weak reasoning to force every inheritance problem into a dominant-recessive pattern?
- How does evidence help us choose the right model?
