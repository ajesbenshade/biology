# Day 14: Problem-Solving Clinic and Review Workshop

## Objectives

- Solve genetics problems with stronger setup, vocabulary, and justification.
- Identify where an error began instead of only fixing the final answer.
- Improve fluency across Punnett squares, pedigrees, probability, and chi-square.
- Rehearse cumulative review moves through two focused clinic rotations before the synthesis workshop.

## Key Terms

- genotype
- phenotype
- expected ratio
- chi-square
- pedigree evidence

## Textbook Content

- Chapter 12 skill repair across inheritance modeling and data interpretation.
- Review of common problem-solving structures used throughout Unit 4.
- Two clinic-style review rotations that reward accurate setup before speed.

## In-Class Activities / Minilabs

1. **Bellwork (8 min):** Diagnose an incorrect Punnett square and explain the setup error.
2. **Mini-Lesson (15 min):** Walk through a mixed inheritance problem from setup to written conclusion.
3. **Guided Practice Block 1 (25 min):** Students complete leveled practice sets by skill strand with teacher checkpoints.
4. **Guided Practice Block 2 (25 min):** Students switch to a second skill strand and complete a second clinic set with one checked solution before moving on.
5. **Debrief (10 min):** Students write one "watch out" note for future problem solving and one process move that helped most.

## Classroom Routine Notes

- Students begin in skill-strand groups based on recent error patterns and rotate only after the teacher approves one fully shown solution.
- The second rotation uses the same clinic routine as the first so students can focus on problem solving instead of learning a new game structure.
- The teacher uses the second rotation to conference quickly with students whose first errors still appear in the setup line.

## Critical Perspective

- Which kinds of genetics errors come from weak math, and which come from weak biological reasoning?
- How does showing work make scientific thinking easier to evaluate?

## Homework / Assignment

- Complete one mixed review set with full work shown.
- Correct any clinic problem you missed and explain why the correction is stronger.

## Exit Ticket / Reflection

Which problem type feels most manageable now, and what changed in your process?

## Teacher Notes Only (not for students)

- Group students by dominant error pattern when possible so feedback stays specific.
- Keep a few challenge problems ready for students who finish early with accuracy.
