# Day 2: DNA Structure and Replication

## Objectives

- Model the structure of DNA with correct vocabulary and strand direction.
- Explain semi-conservative replication and why accuracy matters.
- Use an error-check routine to diagnose common replication mistakes.

## Key Terms

- nucleotide
- complementary base pairing
- semi-conservative replication
- helicase
- DNA polymerase

## Textbook Content

- Chapter 11 coverage of DNA structure and replication.
- Enzyme roles in strand separation, copying, and joining.
- Accuracy checks that prevent weak replication explanations from becoming habits.

## In-Class Activities / Minilabs

1. **Bellwork (8 min):** Sketch a short DNA segment from memory and label the base pairs.
2. **Mini-Lesson (15 min):** Model strand direction, base pairing, and the replication sequence.
3. **Guided Practice Block 1 (25 min):** Build and label a DNA model with a partner.
4. **Guided Practice Block 2 (27 min):** Run a replication simulation and correct one flawed sample model.
5. **Debrief (10 min):** Name one error source and one feature that protects replication fidelity.

## Classroom Routine Notes

- Partners check strand direction before they check enzyme order.
- The class uses the same cue each time it pauses: check direction, check pairs, check enzymes.
- The teacher pulls a short reteach table for students who reverse template and new strands.

## Critical Perspective

- Why is accuracy more important than speed in DNA replication?
- What kinds of evidence show that a model is biologically sound?
