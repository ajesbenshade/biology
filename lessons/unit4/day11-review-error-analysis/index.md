# Unit 4 Day 11 — Review and Error Analysis

## Objectives (Tested)

- Identify and correct common errors in genetics problems
- Strengthen explanations using accurate vocabulary
- Improve data interpretation for quiz/test readiness

## Activities

1. Class-wide misconception review
2. Error-analysis workshop with sample responses
3. Short readiness check

## Assessment (Formative)

- Readiness check and corrected response submission

## Homework (Formative)

- Complete final Unit 4 review sheet

## Teacher Notes Only (not for students)

- Use discussion prompts to reinforce purpose and responsibility in scientific learning.
- Keep grades focused on mastery of content and reasoning.
