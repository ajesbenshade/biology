# Day 10: Lab Skills and Mixed Practice

## Objectives

- Apply codon-table decoding to protein synthesis tasks.
- Solve mixed Punnett square and chi-square problems.
- Label DNA replication structures and identify mutation types.

## Key Terms

- codon table
- chi-square
- leading strand
- lagging strand
- mutation type

## Textbook Content

- Unit 4 cumulative skill integration from Chapters 11–13.
- Emphasis on accuracy, process clarity, and error correction.

## In-Class Activities / Minilabs

1. **Bellwork (8 min):** Complete one mixed-review problem with justification.
2. **Practice Stations (40 min):** Codon decoding, inheritance probability, replication labeling, mutation classification.
3. **Partner Error-Check (15 min):** Trade solutions and annotate corrections.
4. **Peer Teaching (7 min):** Students explain one station concept to a partner.
5. **Wrap-Up (5 min):** One challenge problem synthesis.

## Critical Perspective

- Which mistakes came from calculation versus vocabulary misuse?
- How does careful process checking improve scientific reliability?
