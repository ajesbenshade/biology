# Unit 4 Day 10 — Lab Skills and Mixed Practice

## Objectives (Tested)

- Apply codon table decoding correctly
- Solve mixed Punnett square and chi-square problems
- Label DNA replication structures and enzymes accurately
- Explain mutation type from sequence changes

## Activities

1. Mixed practice stations (codon table, chi-square, replication, mutations)
2. Partner error-check and correction round
3. Exit ticket: one problem from each category

## Assessment (Formative)

- Mixed-practice checkpoint with targeted feedback

## Homework (Formative)

- Finish any incomplete station tasks and submit corrections

## Teacher Notes Only (not for students)

- Emphasize careful observation, precision, and integrity in scientific work.
- Keep scoring criteria strictly tied to textbook objectives.
