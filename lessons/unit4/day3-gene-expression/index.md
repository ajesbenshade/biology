# Day 3: Gene Expression and Protein Synthesis

## Objectives

- Describe transcription from DNA to mRNA
- Describe translation from mRNA to protein
- Use codon language accurately in explanations

## Bell Work (10 min)

How does a cell use DNA information to make proteins?

## Lesson Flow (70 min)

1. Central dogma mini-lesson
2. Guided transcription and translation practice
3. Codon-table decoding activity

## Classwork / Homework

- Protein synthesis practice set

## Assessment

- Exit ticket: Translate a short mRNA sequence
