# Day 9: Biotechnology Data Analysis

## Objectives

- Interpret PCR output data accurately.
- Analyze gel electrophoresis band patterns.
- Write claim-evidence-reasoning responses from biotechnology datasets.

## Key Terms

- PCR output
- gel band pattern
- control sample
- claim-evidence-reasoning
- data interpretation

## Textbook Content

- Chapter 13 data analysis skills for biotechnology investigations.
- Comparative interpretation of multiple evidence sources.

## In-Class Activities / Minilabs

1. **Bellwork (8 min):** Quick PCR and gel interpretation warm-up item.
2. **Guided Analysis (20 min):** Model how to read sample outputs and identify controls.
3. **Small-Group Set (25 min):** Interpret two biotechnology scenarios using provided data.
4. **CER Writing (12 min):** Draft a concise claim-evidence-reasoning response.
5. **Debrief (5 min):** Peer check one evidence statement for precision.

## Critical Perspective

- What makes one dataset more reliable than another?
- Which conclusions are warranted versus overstated from the available evidence?

## Biblical Integration

Truthful scientific communication requires precision, honesty, and responsible interpretation of evidence.

## Homework / Assignment

- Correct missed data-interpretation items.
- Complete one extension analysis problem.

## Exit Ticket / Reflection

What was your strongest evidence for your CER claim today?

## Teacher Notes Only (not for students)

- Prioritize evidence quality and vocabulary accuracy in feedback.
- Encourage students to justify claims with specific data points.
