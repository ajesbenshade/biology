# Day 3: Fossilization and Index Fossils

## Objectives

- Sequence major steps in fossilization and identify conditions that increase fossil preservation.
- Explain how index fossils are used to correlate rock layers.
- Interpret a trilobite minilab data table.

## Key Terms

- fossilization
- sedimentation
- mineralization
- index fossil
- geologic correlation

## Textbook Content

- Chapter 14.2 explains why fossilization is uncommon and requires specific burial conditions.
- Index fossils are species with broad distribution but short time ranges, useful for dating strata comparatively.
- Trilobite examples show how fossil presence can correlate layers across locations.

## In-Class Activities / Minilabs

1. **Fossilization Sequence Cards (15 min):** Put fossilization stages in order.
2. **Trilobite Correlation Minilab (35 min):** Match strata columns A, B, C by index fossil presence.
3. **Evidence Confidence Scale (10 min):** Rank confidence in each correlation claim.

**Sample answers:**
- Typical sequence: death → rapid burial → soft-tissue decay → mineral replacement/impression.
- If Layer A and Layer C both contain the same index trilobite species, they are likely similar relative age.

## Critical Perspective

- Which fossil conclusions are strongest because they are based on direct physical evidence?
- What assumptions are used when inferring large-scale history from incomplete fossil records?
- How does an index fossil method help or limit certainty?
- How should gaps in the record affect the strength of historical claims?
- What alternative catastrophic-burial scenarios are discussed by some interpreters, and how should those claims be evidence-tested?

## Biblical Integration

The fossil record invites humility: it provides real historical clues but not exhaustive detail. Christians can study fossils carefully and still affirm that God is Creator and sustainer of life.

- Answer short response: “Why do index fossils support relative dating rather than exact absolute dates?”
- Add one paragraph comparing two interpretations of fossil order using specific evidence.

## Exit Ticket / Reflection

Name one fossilization requirement and explain why missing that condition reduces preservation likelihood.

- Assess precision of vocabulary and evidence use in lab responses.
