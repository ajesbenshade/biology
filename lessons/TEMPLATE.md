# Lesson Template Standard (Day-Level Pages)

Use this structure for all day-level lessons in Units 4 and 5.

## Frontmatter (required)

```yaml
---
layout: default
title: "Day XX: [Lesson Title]"
permalink: /lessons/unitX/day-XX/
---
```

## Section Order

1. `# Day XX: [Lesson Title]`
2. `## Objectives`
3. `## Key Terms`
4. `## Textbook Content`
5. `## In-Class Activities / Minilabs`
6. `## Critical Perspective`
7. `## Biblical Integration`
8. `## Homework / Assignment`
9. `## Exit Ticket / Reflection`
10. `## Teacher Notes Only (not for students)`
11. `## Navigation`

## Required vs Optional

- Required on all pages:
  - Objectives
  - In-Class Activities / Minilabs
  - Homework / Assignment
  - Exit Ticket / Reflection
  - Navigation
- Recommended where relevant:
  - Key Terms
  - Textbook Content
  - Critical Perspective
  - Biblical Integration
  - Teacher Notes Only

## Writing Rules

- Preserve science content accuracy and textbook references.
- Keep the tone academically rigorous and concise.
- Keep Scripture/worldview integration clear and respectful, not preachy.
- Keep links and images that support instruction.
- Keep navigation links consistent with canonical day URLs.
