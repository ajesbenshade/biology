# Unit 4 DNA Replication Error Menu

Prepared for Dock Mennonite High School Freshman Biology to support careful molecular reasoning, accurate model diagnosis, and responsible stewardship of scientific understanding.

## Purpose

Use this menu during Unit 4 Day 2, "Creator's Code: Build, Flaw, and Restore." Each group should build one flawed DNA model with exactly one hidden error. The goal is not to trick classmates with chaos but to train them to identify disorder by using clear scientific criteria.

## Class Diagnosis Routine

- check direction
- check pairs
- check backbone

## Error Menu

- **Mismatched bases:** one pair is built incorrectly, such as A-C or G-T instead of A-T or C-G.
- **Reversed strand direction:** one strand is oriented the wrong way, so the model no longer shows antiparallel structure.
- **Incorrect backbone linkage:** a sugar-phosphate connection is missing, mislinked, or placed between the wrong parts.
- **Extra nucleotide inserted:** one extra nucleotide appears where it does not belong in the sequence.
- **Missing nucleotide:** one nucleotide is absent, leaving an incomplete strand.
- **Replication copy error:** the new strand contains one non-complementary nucleotide during the replication step.

## Rules For Use

- Each flawed model must contain exactly one hidden error.
- Groups must be able to explain why the chosen flaw is inaccurate using correct vocabulary.
- The receiving group must identify the flaw before changing any pieces.
- Repairs should be justified aloud with evidence from DNA structure and replication rules.

## Teacher Guidance

- Start with easier errors, such as mismatched bases, before assigning directionality or backbone errors.
- If students are guessing, stop the exchange and require them to verbalize the full routine.
- Use the discussion to reinforce that real scientific work values careful diagnosis over quick assumptions.

## Biblical Integration

**1 Corinthians 14:33** reminds students that God is not the author of confusion. The purpose of the Error Menu is to help students recognize when a model departs from the ordered pattern God built into living systems and to correct it responsibly.
