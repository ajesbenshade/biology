# Teacher Guide: Unit 4 Day 3 – Gene Expression and Protein Synthesis

> **For teacher use only.** This guide contains answer keys, lesson scripts, and grading notes not intended for student view.

---

## Scripture Focus

**Colossians 1:16-17** (continued from Day 2)
> "All things were created through him and for him. He is before all things, and in him all things hold together."

Reinforce: The central dogma—DNA → RNA → protein—shows an ordered, consistent information flow. This consistency exists because God upholds creation. We study it with precision and gratitude.

---

## Teacher Objectives

- Explain transcription: DNA is read in the nucleus and copied into mRNA.
- Explain translation: mRNA is read at the ribosome, using codons to assemble amino acids into a protein.
- Use a codon table to predict amino acid sequences from mRNA codons.

**Background for you:** The "central dogma" is just a name for information flow in cells: DNA makes RNA makes protein. Information flows one way (normally). The codon table is universal—nearly every organism uses the same code, which is powerful evidence of a common design.

---

## Key Biblical Integration Points

- "This precise code-to-protein process displays God's order. Every codon maps to one amino acid—it's a language with exact grammar, built into life itself."
- "The universality of the genetic code across all life is a remarkable feature to notice—it invites us to study with humility and precision, and to ask why such consistency exists."

## Materials & Preparation

- Codon table handouts (universal genetic code chart, one per student).
- Transcription/translation flowchart template (optional—for scaffolding).
- Whiteboard space to walk through a worked example.

## Bell Work Answer Key (10 min)

**Question:** Explain how a cell uses DNA information to build proteins.

**Strong answer:**
> "DNA in the nucleus is transcribed into mRNA. The mRNA travels to the ribosome, where it is translated—read three bases (codons) at a time. Each codon corresponds to one amino acid. Amino acids are linked in order to form a protein."

**Simple but acceptable:**
> "DNA is copied to mRNA, which is then read by a ribosome to make a protein."

**Grading:** 5 pts. Award full credit if transcription and translation are both present. Deduct 2 pts if only one step is described.

---

## Lesson Delivery Script (70 min)

### 1. Mini-Lesson: Central Dogma (20 min)

Write on board: **DNA → mRNA → Protein**

Say:
> "DNA stays in the nucleus—think of it as the master blueprint that never leaves the vault. To use the blueprint, a copy is made: that copy is mRNA. The mRNA travels out to the ribosome, which reads it and builds a protein."

**Key distinction:**
- *Transcription* = writing a copy (DNA → mRNA). Happens in the nucleus.
- *Translation* = reading the copy and building (mRNA → protein). Happens at the ribosome.

**Why U instead of T in mRNA?**
> "RNA uses uracil (U) instead of thymine (T). The chemical reason is that RNA is a temporary copy—uracil is slightly less stable than thymine, which is fine for a disposable messenger. It's a built-in system of distinctions."

### 2. Guided Practice: Transcription and Translation (25 min)

**Worked example to walk through on the board:**

| Step | Sequence |
|------|---------|
| DNA template | `3'-T-A-C-G-G-A-A-T-G-5'` |
| mRNA (transcription) | `5'-A-U-G-C-C-U-U-A-C-3'` |
| Codons | `AUG` / `CCU` / `UAC` |
| Amino acids (from codon table) | Met / Pro / Tyr |

Say:
> "AUG is also the start codon—every protein synthesis begins here. Then we read three bases at a time, find the matching amino acid on the codon table, and link them in order."

**Sentence stem for students:** "The DNA sequence ___ is transcribed as ___ mRNA, which translates to the amino acid ___."

### 3. Codon Table Lab (10 min)

Provide 3–4 short mRNA sequences. Students decode using the codon table.

**Sample sequences and answers:**

| mRNA | Codons | Amino Acids |
|------|-------|------------|
| `AUGUUUAGA` | AUG / UUU / AGA | Met / Phe / Arg |
| `AUGCCGUAA` | AUG / CCG / UAA | Met / Pro / Stop |
| `AUGCAGCUU` | AUG / CAG / CUU | Met / Gln / Leu |

### 4. Debrief (5 min)

Ask: "What would happen if one base in the mRNA changed?"

Expected answer: "A different amino acid might be inserted, which could change the protein's shape and function." (Point mutation → potential protein change.)

Say:
> "That's why precision in this process matters—and why the built-in proofreading systems we discussed yesterday are so important."

## Exit Ticket Answer Key

**Question:** Translate a short mRNA sequence and justify one codon choice.

**Example prompt:** Translate `AUGCCCUGA`

**Answer:** AUG = Met (Start), CCC = Pro, UGA = Stop. Protein = Met-Pro (then synthesis ends).

**Justification example:** "I used UGA as a stop codon because the codon table shows UGA does not code for any amino acid—it signals the ribosome to release the protein."

**Grading:** 5 pts. Full credit: correct translation + codon table citation as reasoning. Deduct 2 pts for wrong amino acid without reasoning.

## Homework Grading Key

| Assignment | Points | Key Criteria |
|-----------|--------|-------------|
| Protein synthesis practice set | 15 pts | Accurate transcription (T→U substitution); correct codon → amino acid matching |
| Evidence-based paragraph on codons → amino acids | 5 pts | Must reference codon table as evidence; explain how 3-base codes determine amino acid identity |

**Strong paragraph example:**
> "Each three-base sequence of mRNA (a codon) corresponds to exactly one amino acid, as shown in the codon table. For example, the codon AUG always specifies methionine, and UGA is always a stop signal. This precise, universal code allows the same genetic language to be read the same way in nearly every organism—a consistency that reflects ordered design rather than chance."

## Common Confusions & Fixes

| Confusion | Fix / What to Say |
|-----------|------------------|
| "RNA and DNA are the same thing" | "Related but different. DNA is double-stranded, uses thymine, and is permanent. RNA is single-stranded, uses uracil, and is temporary." |
| "The ribosome reads DNA directly" | "No—RNA polymerase reads DNA and makes mRNA. The ribosome only reads mRNA. DNA stays in the nucleus." |
| "Every mutation changes the protein" | "Not always. The genetic code has redundancy—some codons code for the same amino acid. A change in the third base of a codon often doesn't change the amino acid (called a silent mutation)." |

---

## Differentiation

- **Advanced:** Analyze the effect of one point mutation on an amino acid sequence. Does the mutation change the protein? Is it silent or missense? Research one real genetic disease caused by a specific point mutation (e.g., sickle cell: GAG → GUG).
- **Struggling learners:** Use a scaffolded codon chart with color-coded codon families. Provide step-by-step index cards for transcription and translation procedures.
- **ELL support:** Provide bilingual codon table key terms if available. Visual flowchart of the central dogma with labeled arrows.
