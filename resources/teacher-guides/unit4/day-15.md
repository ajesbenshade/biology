# Teacher Guide: Unit 4 Day 15 – Biotechnology CER and Mixed Practice

## Scripture Focus

**Ephesians 4:25**

Use the writing day to connect truthful speech with precise scientific explanation.

## Teacher Objectives

- Build one strong biotechnology CER model with the class.
- Give students real independent writing time.
- Use mixed practice to identify final Chapter 13 weak spots before synthesis.

## Materials & Preparation

- Shared dataset for class CER
- Individual CER prompt
- Short mixed biotechnology review set

## Routine and Logistics Notes

- Students write independently before peer feedback begins.
- Partner checks focus on evidence specificity before style.
- Teacher conferences should end with one concrete Day 16 target.

## Bell Work Answer Key

Strong revisions improve a weak evidence sentence with a specific control or band comparison.

## Lesson Delivery Script (85 min)

1. **Bellwork (8 min):** Improve a weak evidence sentence.
2. **Mini-Lesson (15 min):** Build a class CER from one shared dataset.
3. **Guided Practice Block 1 (25 min):** Students write one full CER independently.
4. **Guided Practice Block 2 (25 min):** Complete mixed biotechnology practice and revise one CER sentence.
5. **Debrief (10 min):** Mark one Chapter 13 strength and one review target.

## Exit Ticket Answer Key

Strong responses name one clear Chapter 13 strength and one concrete skill still needing review.

## Planning Note

Use today's work to set Day 16 conference priorities and identify which synthesis supports students will need next.

## Prayer Close

Ask for wisdom, honesty, and confidence rooted in preparation rather than guessing.
