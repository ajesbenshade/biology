1) Chi-Square Practice
Problem 1 (3:1, total 160):
Expected = 120 dominant, 40 recessive
χ² = ((118-120)^2/120) + ((42-40)^2/40)
χ² = 0.0333 + 0.10 = 0.1333
df = 1
Interpretation: very good fit to 3:1

Problem 2 (9:3:3:1, total 320):
Expected = 180, 60, 60, 20
χ² = (16/180) + (36/60) + (64/60) + (36/20)
χ² ≈ 0.0889 + 0.6 + 1.0667 + 1.8 = 3.5556
df = 3
Interpretation: generally acceptable fit (compare to critical values used in class)

Problem 3 (1:1, total 200):
Expected = 100, 100
χ² = (25/100) + (25/100) = 0.5
df = 1
Interpretation: good fit to 1:1

3) Replication Labeling
- DNA polymerase extends new DNA only 5' -> 3'.
- Lagging strand forms Okazaki fragments because template orientation is opposite fork movement.
- Ligase seals phosphodiester backbone gaps between fragments.
- Semi-conservative: each daughter DNA has one old strand + one new strand.
